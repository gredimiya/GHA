# GHA

Guess Hide Anime

## Règles du jeu

* Une page MyAnimeList (MAL) choisis au hazard avec 90% des infos cacher
* Chaque indice révèle 10% d'infos
* Calcul des points :
    * 10 Points si aucun indice
    * 8 Points pour 1 indices
    * 7 Points pour 2 indices
    * 6 Points pour 3 indices
    * 5 Points pour 4 indices
    * 4 Points pour 5 indices
    * 3 Points pour 6 indices
    * 2 Points pour 7 indices
    * 1 Points pour 8 indices
    * Retire 2 Points pour le choix de l'élement à montrer
* Pas de temps limite en solo
* Possibilité de limité les pages en fonction de :
    * Note de l'anime
    * Année de sortie
    * Classement popularité
    * Classement visionnage
    * Genre
    * Présent dans sa liste MAL
* Possibilité de choisir le nombre de page à jouer
